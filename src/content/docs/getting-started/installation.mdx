# Installation

Follow these steps to install TinyDocs and get your documentation site up and running.

## Prerequisites

Before you begin, ensure you have the following installed:
- Node.js 14.16 or higher
- pnpm (recommended) or npm

## Create Your Site

### Using the Template

1. Click the "Use this template" button on the GitHub repository
2. Create a new repository from the template
3. Clone your new repository:

```bash
git clone https://github.com/yourusername/your-docs-site
cd your-docs-site
```

### Install Dependencies

```bash
pnpm install
```

Or with npm:

```bash
npm install
```

## Start Development Server

Start the local development server:

```bash
pnpm dev
```

Your site will be available at `http://localhost:4321`

## Next Steps

Continue to [Configuration](/docs/getting-started/configuration) to customize your site.
