# Introduction

Welcome to TinyDocs! This is a minimal documentation framework built with Astro.

## What is TinyDocs?

TinyDocs is designed to help you create beautiful, fast, and accessible documentation sites with minimal effort. It's built on top of Astro and uses MDX for content authoring, giving you the power to create interactive documentation with custom components.

## Core Features

### Minimal and Fast
- Built with Astro for optimal performance
- Minimal JavaScript by default
- Static site generation for blazing fast load times

### Developer-Friendly
- TypeScript support out of the box
- Tailwind CSS for easy styling
- MDX support for interactive documentation

### Organized Navigation
- Sidebar navigation with collapsible sections
- Table of contents for each page
- Clear hierarchy for your documentation structure

### Accessible
- System, dark, and light mode support
- Keyboard navigation
- Semantic HTML standards
