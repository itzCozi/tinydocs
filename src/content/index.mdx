# Welcome to TinyDocs

TinyDocs is a minimal documentation framework built with [Astro](https://astro.build/). Perfect for developers who want a clean, fast, and customizable documentation site without the bloat.

## Key Features

- **Minimal and customizable:** Clean foundation to build upon and meet your unique needs
- **Fast by default:** Leverage the power of Astro, Tailwind CSS, and Markdown to create fast, lightweight web pages
- **Accessible to all:** System, dark, and light mode support. Keyboard navigation and semantic HTML standards
- **SEO-friendly:** Sitemap and Open Graph protocol support out of the box
